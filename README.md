Miniblox.io Custom Crosshair

Miniblox.io için geliştirilen Custom Crosshair uzantısı, nişangahınızı tamamen kişiselleştirmenizi sağlar.

✨ Özellikler
🎯 Özelleştirilebilir crosshair (boyut, renk, kalınlık)
🌈 Renk seçici
💾 Ayar kaydetme
⚡ Hafif ve hızlı
🚀 Kurulum (Chrome / Edge - Developer Mode)
1. Dosyayı İndir
Bu projeyi ZIP olarak indir veya klonla:
git clone https://github.com/YusiGeXD/miniblox-crosshair-extension.git
ZIP indirdiysen klasöre çıkart (extract).
2. Tarayıcıda Uzantılar Sayfasını Aç

Chrome için:

chrome://extensions/

Edge için:

edge://extensions/
3. Developer Mode Aç
Sağ üst köşede bulunan "Geliştirici Modu (Developer Mode)" seçeneğini aktif et.
4. Uzantıyı Yükle
"Paketlenmemiş öğe yükle (Load unpacked)" butonuna tıkla
Az önce çıkardığın proje klasörünü seç
5. Tamam 🎉

Uzantı artık aktif! Miniblox.io açtığında crosshair özelleştirmesi çalışacaktır.

🛠️ Kullanım
Oyuna gir
Ayar panelini aç
Crosshair ayarlarını değiştir
Anında etkisini gör
⚠️ Sorun Giderme

Uzantı görünmüyorsa:

Developer Mode açık mı kontrol et
Doğru klasörü seçtiğinden emin ol (içinde manifest.json olmalı)

Çalışmıyorsa:

Sayfayı yenile
Uzantıyı kapat/aç
